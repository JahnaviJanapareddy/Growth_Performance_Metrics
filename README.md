# 📊 Growth Performance Metrics Dashboard

## 📌 Project Overview
This project presents a data-driven Power BI dashboard designed to analyze and monitor the performance of a company’s Growth Department across key subscription metrics. It focuses on evaluating customer acquisition, retention, revenue stability, and cost efficiency across both B2B and B2C segments.

The dashboard enables data-informed decision-making by providing clear insights into subscription trends, churn behavior, and overall business health.

---

## 🎯 Objectives
- Monitor monthly revenue trends and detect performance fluctuations  
- Track active subscriptions to evaluate growth and retention  
- Analyze churn rates to identify customer loss patterns  
- Measure marketing spend efficiency  
- Compare B2B vs B2C performance for strategic decision-making  

---

## 📂 Dataset Description
The project uses a structured dataset consisting of 5 key tables:

| Table Name        | Description |
|------------------|------------|
| Customers        | Customer demographics and acquisition details |
| Subscriptions    | Subscription lifecycle and pricing information |
| Revenue          | Transaction-level revenue data |
| Marketing Spend  | Channel-wise marketing investments |
| Funnel           | Conversion funnel metrics (visitors → leads → trials → paid users) |

---

## 📊 Key Metrics (KPIs)
- Revenue (Current vs Previous Month)  
- Active Subscriptions  
- Churned Subscriptions  
- Marketing Spend (MoM Comparison)  
- Revenue by Segment (B2B vs B2C)  

These KPIs were implemented using DAX calculations to ensure dynamic and time-aware analysis.

## 📈 Dashboard Features

## 🧠 Business Insights
- Revenue remained relatively stable over time  
- Mid-year churn increase highlighted potential retention issues  
- Segment-level insights (B2B vs B2C) supported better resource allocation  
- Spend tracking enabled improved cost control  

Overall, the dashboard helps drive sustainable growth through continuous performance monitoring.
